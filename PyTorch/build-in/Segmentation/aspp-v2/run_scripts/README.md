## 参数介绍

参数名 | 说明 | 默认值
-----------------|-----------------|-----------------
model_name |模型名称。 | --model_name DenseFuse
data_path | 数据集路径。 | --data_path ../../../dataset/VOCdevkit
num_classes |分类数量。 | 20
device | 设备名称。 | sdaa
batch_size | 训练批次大小。 | 4
epochs| 批量大小。 | 1
lr| 学习率。 | 0.0001
amp | 是否开启混合精度。 | True
