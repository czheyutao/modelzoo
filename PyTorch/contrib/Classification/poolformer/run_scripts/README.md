## 参数介绍

参数名 | 说明 | 示例
-----------------|-----------------|-----------------
data_dir |数据集存储路径。 | /data/teco-data/imagenet
model |模型名称。 | --model poolformer_s12
epochs | 训练轮次。 | --epochs 100
warmup-epochs | 预热轮次。 | --warmup-epochs 0
cooldown-epochs | 冷却轮次。 | --cooldown-epochs 0
amp | 自动混合精度。 | --amp
